# CPEG 208 — Microprocessors

---

## Overview

This repository contains coursework for **CPEG 208: Microprocessors**, covering PIC16F84A assembly language programming using MPLAB X IDE and Proteus simulation. Topics include GPIO configuration, bank switching, delay loop timing, interrupt handling, and cycle-accurate instruction analysis.
